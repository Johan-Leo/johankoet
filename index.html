about Me

My name is Johan Koet,

I'm a 28 years old aspiring front-end developer from the Netherlands. The last few years I've been discovering my creative side more and more. It's something that has always been there, but somewhere in my teenage years I slowly stopped using it. A few years back I discovered a little talent for drawing. I really enjoyed doing it and I thought about pursuing a creative career.

I started a year course in graphic design, in which I found out that I really liked making illustrations in Adobe Illustrator. I did some design work, mostly for family, but something was missing...

During the course we also got a quick word press introduction, which peaked my interest in how websites are actually made. We never touched the coding side, but it got mentioned. So after the course I started to dive into (front-end) web development. Then something clicked for me: It was the perfect balance between doing creative and technical work. It's like solving a creative puzzle that turns into something pleasing to look at, which at the same time has to be something functional.

I started out learning about the CSS and HTML basics, and then I dove into JavaScript the last few months. I'm a perfectionist and I always want to know exactly how things work, which makes me a very thorough student. The prove of that is in a personal project that I've been working on the past few months, which I will tell you all about at the bottom of this page. But first you get so see a small collection of the illustrations that I've made.

coding Koala

As I mentioned earlier: I'm very thorough when it comes to learning. During the online courses I made a lot of notes, which all ended up being stacked on top of each other on the Udemy website, which is where I follow all my courses. It was hard to keep track of all the notes, which made it difficult to quickly check some specific notes. On top of that it's a pretty slow and boring way of refreshing your memory on certain topics, so I figured it was time for my very first project.

This project would be: Coding Koala. A multiple choice coding quiz website, where I'd turn all the notes I've made during the development courses into multiple choice questions. I made the design in Adobe Illustrator and then I started coding.

Sometimes I plateaued and couldn't figure out how to code something, mostly because I hadn't learned about DOM manipulation yet. But then I'd just continue the course, and then after some small follow along projects introducing DOM manipulation I finally got it. It was great to see the website come to life, and especially to see it become functional!

It still is a work in process, but I'm really happy with how it's turning out. I'm slowly adding new things now, and then eventually it will become a very functional quiz website. And for now it's already more than enough for me to go through my notes in a cool, interactive way.

Feel free to check it out by clicking on the little koala!
